# rat_quickdb 已有能力索引

本文档记录项目已有的模块、工具函数、设计模式及其位置，便于实现新功能时查找已有实现，避免重复造轮子。

## 核心入口

| 能力 | 位置 | 说明 |
|------|------|------|
| `init()` | `src/lib.rs` | 初始化 i18n 系统，必须在使用前调用 |
| `generate_object_id()` | `src/lib.rs` | 生成 MongoDB 风格的 ObjectId |
| `process_data_fields_from_metadata()` | `src/lib.rs` | 处理数据库返回的原始数据（JSON/Bool/DateTime 转换） |
| `lock_global_operations()` | `src/lib.rs` | 锁定全局操作，禁止添加新数据库配置 |
| `is_global_operations_locked()` | `src/lib.rs` | 检查全局操作是否已锁定 |
| `get_info()` | `src/lib.rs` | 获取库版本和名称信息 |
